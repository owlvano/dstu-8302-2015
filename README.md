# dstu-8302-2015

**ДСТУ 8302:2015**  
CSL-стиль для **Zotero / citeproc**, адаптований під реальні вимоги:
- українських закладів вищої освіти;
- фахових журналів;
- наукових конференцій;
- дисертаційних робіт.

---

Цей проєкт перебуває у стані **work in progress (WIP)**.

Стиль проходить поступову перевірку на відповідність:
- формальним вимогам CSL;
- положенням ДСТУ 8302:2015;
- практичним вимогам редакцій наукових видань.

Можливі подальші зміни, уточнення та виправлення форматування.

## Про репозиторій

Цей репозиторій є **fork** оригінального проєкту  
https://github.com/tetiuk/dstu-8302-2015

Оригінальний CSL-стиль було проаналізовано та модифіковано з метою усунення практичних проблем, що виникають під час подання бібліографії до редакцій наукових видань та спеціалізованих вчених рад.

## Структура репозиторію

У репозиторії наявні два CSL-файли:

### `dstu-8302-2015-clean.csl`  
**DSTU 8302:2015 (clean academic)**

Рекомендовано для:
- дисертацій;
- авторефератів;
- наукових звітів;
- українських наукових конференцій.

Особливості:
- анотація не виводиться (відповідає ДСТУ 8302:2015);
- не використовується курсив для назв журналів;
- числове цитування у тексті;
- підтримка DOI та URL;
- орієнтація на академічне використання.

### `dstu-8302-2015-alphabetical.csl`  
**DSTU 8302:2015 (alphabetical)**

Рекомендовано для:
- журналів, що вимагають алфавітний список літератури;
- міжнародних та фахових наукових видань.

Особливості:
- алфавітне сортування джерел;
- числове цитування у тексті;
- анотація не виводиться;
- підтримка одночасного виведення DOI та URL;
- редакційна адаптація ДСТУ 8302:2015.

## Основні відмінності від оригінального стилю

У порівнянні з оригінальним стилем `dstu-8302-2015`:
- вилучено виведення анотації з бібліографічного опису;
- усунено курсив у назвах журналів;
- покращено підтримку електронних ресурсів (DOI, URL, дата звернення);
- додано альтернативну версію з алфавітним сортуванням;
- виправлено формальні помилки валідації CSL.

## Використання інструментів штучного інтелекту

Під час аналізу та модифікації CSL-стилю було застосовано **ChatGPT 5.2** як допоміжний інструмент для:
- виявлення невідповідностей між оригінальним стилем та вимогами ДСТУ 8302:2015;
- аналізу логіки сортування та форматування бібліографічних елементів;
- формування рекомендацій щодо структурних змін CSL.

Остаточні рішення щодо внесених змін приймалися автором репозиторію.

## Встановлення стилю в Zotero

1. Завантажте Zotero з офіційного сайту:  
   https://www.zotero.org

2. Відкрийте Zotero та перейдіть до:  
   **Правка → Налаштування → Цитування → Стилі**

3. Натисніть **[ + ]** та оберіть потрібний `.csl` файл:
   - `dstu-8302-2015-clean.csl`, або
   - `dstu-8302-2015-alphabetical.csl`

4. Підтвердьте вибір. Стиль з’явиться у списку доступних.

5. У текстовому редакторі (Word / LibreOffice) оберіть відповідний стиль цитування.

## Зауваження

- Поле **Abstract** у Zotero не використовується при формуванні бібліографії.
- Для одного документа рекомендовано застосовувати лише один CSL-стиль.
- Для дисертаційних робіт рекомендовано використовувати версію *clean academic*.
